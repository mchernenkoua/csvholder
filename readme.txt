Project description

Технологии

    H2Database база данных (Удобна для тестирования) 
    Hibernate как маппер обьектов в базу данных и работа с запросами к базе
    Gson библиотека для работы с настройками
    JavaFX для построения интерфейса приложения 

Задание

Нужно написать простую программу в которой пользователь сможет настроить сохранение csv файла в базу данных. 
Правила сохранения следующие: 

    Пользователь выбирает файл
    Пользователь выбирает название таблицы
    Программа парсит xls файл и выводит список всех не пустых колонок в одну колонку и дублированные названия полей в таблице во второй колонке. 
    Пользователь может отредактировать названия полей таблицы, но не колонок файла
    При нажатии на кнопку сохранить программа создает таблицу - если ее нет и сохраняет все данные в базу, 
		берет построчно значения полей и с помощью insert сохраняет их в БД
    Данные о маппинге должны сохраниться в json файл
